# Definition of Done

The implementation is complete when:

- Angular 20 standalone app builds successfully
- Dark/Light theme toggle works and persists
- Work page renders projects from typed metadata
- Project modal is fully accessible and supports deep linking
- Resume page is print-friendly
- SEO Title/Meta updates per route
- JSON-LD structured data exists
- Lighthouse performance is strong
- Build works for S3 + CloudFront deployment
