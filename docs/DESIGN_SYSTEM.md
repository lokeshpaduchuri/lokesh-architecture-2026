# Design System Guidelines

Typography:
- Clean sans-serif
- Strong hierarchy
- No decorative fonts

Spacing:
- Consistent vertical rhythm
- Mobile-first layout

Color:
- Dark/Light toggle
- Neutral base with subtle accent
- Avoid loud colors

Motion:
- CSS transitions only
- No excessive animation
- Hover lift on cards
- Subtle fade-in on section entry

UI Tone:
- Professional
- Architect-level
- Structured
