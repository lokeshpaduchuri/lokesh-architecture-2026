# lokesh-architecture-2026

Modern “resume on the web” personal site for Lokesh Paduchuri, built with Angular 20.  
Focused on *what I do* and *how I do it* — with modal-based case studies, strong UX polish, SEO fundamentals, and performance-first delivery.

---

## ✨ Highlights
- Angular 20 (standalone components + router)
- Resume-style layout + print-friendly `/resume`
- Work grid with **modal case studies**
- Deep linking: `/work?project=slug` opens a specific project modal
- Dark/Light theme toggle (respects system preference + persists in `localStorage`)
- Fast by default: responsive images, lazy loading, minimal JS
- SEO: per-route Title/Meta + JSON-LD structured data

---

## 🧱 Tech Stack
- Angular 20
- Tailwind CSS (recommended) or SCSS
- Motion: CSS transitions + keyframes (optional: Motion One for scroll reveals)
